Your First Script

Let's write a simple UI.

main.js 

                            
    import { uiDB, uiManager } from "./@easy/uiManager.js";
    import { world } from "@minecraft/server";
    import { uiBuilder } from './@easy/uiBuilder';

    uiManager.registerUIs("testmeow|Leaf/testmeow"); // Register the UI
    uiManager.registerBuilder(uiDB.uis.testmeow, "§e", (player) => { // Please ignore the §e
        return new UI("§e") // Here too
            .setTitle("Meow UI")
            .addHeader("§l§eTest UI")
            .addLabel("§7This is a test UI with an item icon on a button.")
            .addLabel("This is a test UI with an item icon on a button.")
            .addButton("gravel", "minecraft:stone", "middle", false)
            .setAction(player, (res) => {
            if (res.canceled)
                return;
            if (res.selection === 0) {
                player.sendMessage("You pressed the gravel button!");
            }
        });
    });

Save this and try it on Minecraft.

API Reference

Detailed documentation for all available EasyAPI.

Executor API

The core API for script execution and console interaction.

`executor.print(value)`

Prints a value to the RYOS console.

executor_print_example.lua 
123
                            
executor.print("This is a message.")
executor.print(123)
executor.print(true)

UIManager API

Provides functions to create and manage in-game user interfaces.

ui_manager_example.ts 
123456789101112131415
                            
// Example using UIManager (TypeScript-like)
uiDB.registerUIs("TestMenu | test/menu");

uiManager.registerBuilder(uiDB.uis.test.menu,
    (response, player) => {
        switch (response.selection) {
            case 0:
                player.sendMessage("S");
                break;
            case 1:
                player.sendMessage("W");
                break;
        }
    }
);

Player API

Access player-specific information and actions.

`player.getName()`

Returns the name of the current player.

`player.sendMessage(message)`

Sends a message to the player.

UI Creation Example

An example of creating a simple UI with buttons.

simple_ui.ts 
                            
uiDB.registerUIs("MyMenu | MainMenu");

uiManager.registerBuilder(uiDB.uis.MainMenu,
    uiBuilder.action.button("Click Me"),
    uiBuilder.action.button("Close Menu")
);

-- When "Click Me" is pressed, a response can be handled.
-- This part would typically be handled in the builder's response function.
